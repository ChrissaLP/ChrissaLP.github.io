Real Estate Database

×

I built a real estate database to help a potential buyer track residential properties of interest, track appointments and store other useful information (such as listing agent contact details, information on the different localities and their respective tax rates). Stored procedures were included to insert, update, and delete records in the database.

I used SQL Server for the underlying database and Java to create the Graphical User Interface (with the help of NetBeans).
